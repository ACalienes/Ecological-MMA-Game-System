**Status:** Draft / unpublished staging. Merge target: [coverage-map.md](coverage-map.md). Not in the built site.
**Method:** 4 parallel web-research passes (ecological/CLA, positional-by-position, defensive/escape, submission/leglock/scramble). ~90 raw candidates → deduped + cross-referenced against our 36 built games and the 70 coverage-map proposals below.

## 1. The headline find — a free, on-methodology games library

**Slime Mold Grappling Club** — `https://slimemoldgrappling.com/games/index.html` (+ `/lesson_plans/`).
A public, fully-specced **ecological / constraints-led** BJJ games catalog in the **Greg Souders / Standard Jiu-Jitsu** lineage (some games credited to Rob Cole / BJJ Globetrotters). Each game gives start + roles + win/loss, often with explicit thresholds and leglock safety caps. This is the single richest source we've found and the agents only sampled it.

**Action:** deep-mine SMGC game-by-game in the next research pass (it maps almost 1:1 onto our coverage-map gap list). Credit the creators in `reference/sources.md`; re-express mechanics under our names.

Other strong hubs: `grapplerscollective.com/learn/constraints-led-approach` (CLA framework + grip/pass/escape games w/ 3s thresholds), `grapplearts.com` (Kesting — turtle-seatbelt + RNC/back-escape designs), `defensivebjj.com` (Mihkelson defensive postures — IP-flagged), `sonnybrown.net` (Souders/Bath/Biernacki interviews), `bjjcoach.substack.com` (Danaher class formats; a 6-game triangle progression).

---

## 2. The honest finding (validates our threshold house rule)

All four passes independently concluded: **the public BJJ-games corpus almost always leaves the win as "reset on event" or "as long as possible," not a stated time/count.** Explicit thresholds were rare — only a handful (mount-maintain 10–60s, turtle-seatbelt "dislodge the sternum," Ankle Wars 2s, no-hands-pass 3s, EBI fastest-escape scoring). So our locked **threshold house rule** and **observable-non-stall win** are a genuine differentiator, not repackaging. Don't claim sourced thresholds where the source was vague.

## 3. Merge against the coverage map

Legend: **✓CONFIRMS** = source-validates a game we already built or proposed (borrow its concrete start/threshold). **NEW** = fills a gap not on the map, worth adding. **REFINES** = splits one proposed game into several sharper ones.

### Per-position GROUND (coverage map F–I) — heavily confirmed
- **✓CONFIRMS our built escapes** (mount/side/back): SMGC + Grapplearts + Pioneer all run the same mount/side/back escape games; back-escape's "turn to face by any means" + seatbelt start matches what we just built. Borrowable thresholds: mount-maintain 10s→scale, back 2-min×5.
- **✓CONFIRMS maintenance proposals** (#36–38 mount/side/back ride): "Stay Mounted" (10s baseline), "Hold Side Control," "Keep the Back" (seatbelt+hooks) — all real, all event-or-short-count wins.
- **✓CONFIRMS passing proposals** (#31–33): seated-guard pass-vs-sweep, open-guard pass-to-side, half-guard top free-the-leg. Concrete win = reach stable side control / chest-to-chest.
- **✓CONFIRMS submission-offense proposals** (#47–51): SMGC "Maintain Arm Isolation" (mounted armbar), "Submission Hunting" (back, RNC, **constraint: top may not insert 2nd hook**), head-arm/darce/kimura games. Mounted-armbar win = both bottom elbows escape vs top keeps them pinned.
- **NEW — Triangle as a 6-game LADDER** (bjjcoach.substack 6 games; SMGC arm-in/arm-out, causing-posts, punishing-posts, maintain-finish). A *single submission decomposed into a progression of games* (force posts → hook the post → isolate arm → lock → finish). **This is a template idea**: build our submission games as ladders, not one page. High value for the submission-offense wave.
- **NEW — North-South Kimura trap** (control-hub game): top maintains kimura grip through defensive reactions vs bottom frees the arm. Fills the empty north-south bucket (map #40 was only a generic "ride").

### Guard retention/recovery (map #44) — REFINES into several
- The single "Guard Retention" proposal should **split by pass type**: knee-slice recovery, toreando recovery, leg-drag recovery, continuous side-to-side retention (60s), belly-up vs standing, recover-from-near-passed (compromised start). Each is a clean asymmetric isolation game. (RollBliss, heavybjj, yourbjjguide.) Strong, cheap wins.

### Turtle & survival (map #34 + the survival gap) — NEW cluster
- **Turtle seatbelt game** (Grapplearts): top takes back+chokes vs bottom **dislodges the chest/sternum connection** — a crisp, observable win we should adopt verbatim-as-mechanic.
- **Turtle escape: BJJ vs wrestling exits** (Granby/sit-out to guard OR stand-up).
- **Turtle "fortress" hand-fight / hook-denial**, **seated-turtle back defense**, **side-lying travel posture** — re-expressed Mihkelson postures. **IP: do NOT use his names** (Running Man, Grilled Chicken, Panda, Baby Bridge, Sitting Turtle); describe the structure.

### Submission DEFENSE (map J #53–55) — CONFIRMS, with safety
- Choke defense (RNC hand-fight; triangle posture-and-pull-out), joint-lock defense (armbar connect-and-escape / hitchhiker), leglock defense (clear the knee line, hide the heel). All real games; **leglock versions need the slow/controlled safety cap** the sources stress.

### Leglock OFFENSE (map #52) — NEW concrete designs
- **Capture the Foot** (ashi, no finishing — control-only retention), **ankle-lock finishing mechanics** (cooperative, tap-early), **cross-ashi/saddle maintain control** (no heel exposure), **shin-to-shin → SLX entry**. SMGC gives the safety-capped structure we'd want for any leg game.

### Transitions / scrambles (map K #56–60) — CONFIRMS + NEW
- **NEW — Single-leg "access the hip"** (3 progressive versions, early bans the takedown) — a clean takedown-to-pass entry game (map #59).
- **NEW — "Cover the hips"** sprawl-to-back/chest-to-back consolidation (map #60 sprawl-to-top).
- **NEW formats (not games): EBI-style overtime scoring** (fastest-submission / fastest-escape from a locked start) and **King of the Hill** rotation. Treat these like Positional Sparring — **a FORMAT layered over our games**, not new game pages. (IP: "EBI"/"Eddie Bravo Invitational" trademarked; "overtime rounds" is the generic term.)
- **NEW — constraint variants worth a game each**: no-hands-on-legs pass (3s hold win), no-knees top player (forces wrestle-up), pass-or-reset (punishes retreating), 30-second-kill (passer must finish in 30s).

## 4. What this changes for the build

1. **Deep-mine Slime Mold Grappling Club next** — it likely supplies concrete, on-methodology starts + thresholds for most of the proposed per-position ground games, accelerating the whole ground/submission wave.
2. **Build submission-offense as LADDERS** (the triangle 6-game model), not single pages — more pedagogically sound and more games per position.
3. **Split the single "Guard Retention" proposal into ~5 pass-type recovery games** — cheap, high-value coverage.
4. **Add a turtle/survival sub-cluster** (seatbelt dislodge, BJJ-vs-wrestling exits, hook-denial fortress, seated-turtle back defense) — re-expressing Mihkelson structures under our own labels.
5. **Add north-south kimura-trap, single-leg hip-access, cover-the-hips** to close the empty north-south + transition buckets.
6. **Log EBI-overtime scoring + King-of-the-Hill as FORMATS** (like Positional Sparring), not games.
7. **Keep the threshold house rule front and center** — it's our edge over this corpus; every borrowed game gets our clock + observable non-stall win.
