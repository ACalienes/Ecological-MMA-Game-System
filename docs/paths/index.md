# Learning Paths

Guided, ordered journeys through the games. Each path stacks games in dependency-correct order so every step builds on skills you already have. New to the system? Start with **Foundations First** — it's free.

<div class="grid cards" markdown>

-   **[Foundations First](foundations-first.md)** ⭐

    The free starter path — build the defensive shell and range awareness every other path assumes.

-   **[Striking Offense](striking-offense.md)**

    Isolate each weapon, then learn to land combinations and sustain pressure toward a finish.

-   **[Closing the Distance](closing-the-distance.md)**

    Win the circle and convert open-space control into clinch and takedown entries.

-   **[The Wall Game](wall-game.md)**

    Establish, exploit, escape, and cycle the wall pin — the vertical control battle.

-   **[The Ground Game](ground-game.md)**

    Access, control, and finish from top; escape, recover, and stand up from bottom.

-   **[The Complete Fight](complete-fight.md)**

    The capstone — thread every phase from first touch to a finish.

</div>
