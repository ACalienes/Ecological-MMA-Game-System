    Games by Environment

    Skill Isolation

    Constrained games that develop single skills in isolation.

    Defensive

    Game Difficulty Core Problem
    Parry the Straight Beginner Deflect straight punches with hands
    Tight Block Beginner Absorb strikes using tight guard
    Slip the Straight Beginner Evade straight punches with head movement
    Evade the Punch Intermediate Match evasion type to punch type

    Offensive (Weapon Isolation)

    Game Difficulty Core Problem
    Lead Hand Offense Beginner Land strikes using only lead hand
    Rear Hand Offense Beginner Land strikes using only rear hand
    Lead Leg Offense Intermediate Land kicks using only lead leg
    Rear Leg Offense Intermediate Land kicks using only rear leg

    Open Space — Striking

    Standing games focused on striking offense and defense.

    Game Focus Difficulty Core Problem
    Close-Range Defense Defensive Intermediate Layer multiple defensive solutions
    Land the Target Offensive Intermediate Read defense, setup, land predetermined target
    Sustained Offense Offensive Intermediate Maintain rhythm through defensive attempts
    Touch and Don't Get Touched Combined Beginner Range awareness and timing literacy

    Open Space — Wrestling

    Standing games focused on space control and takedowns.

    Game Focus Difficulty Core Problem
    Winning the Circle Combined Beginner Space domination through pressure
    Boundary Trigger Combined Intermediate Recognize clinch opportunity at boundary
    Pressure to Takedown Offensive Advanced Chain strikes to clinch to takedown
    Pressure to Wall Offensive Advanced Chain strikes to wall control for grinding
    Takedown Defense Defensive Intermediate Sprawl and recover from shot attempts
    Clinch Denial Defensive Intermediate Prevent clinch establishment
    Open Space Takedown Combined Intermediate Execute takedowns without wall assistance

    Wall

    Games where the wall is the dominant constraint.

    Game Focus Difficulty Core Problem
    Wall Control Offensive Intermediate Establish and maintain vertical pin
    Wall Escape Defensive Intermediate Break pin and create space
    Wall Pin to Ground Offensive Intermediate Transition from wall to ground control
    Wall Grinding Offensive Intermediate Deal damage from wall position
    Stand-Up Loop Combined Advanced Manage cyclical stand-up and re-pin dynamics

    Ground

    Games focused on horizontal grappling positions.

    Game Focus Difficulty Core Problem
    Ground Access Offensive Intermediate Pass guard and advance position
    Ground Control Offensive Intermediate Maintain position through escape attempts
    Ground Finish Offensive Advanced Finish via strikes or submission
    Ground Escape Defensive Intermediate Escape from under dominant position
    Guard Recovery Defensive Intermediate Recover guard when being passed
    Ground to Standing Defensive Intermediate Get back to feet from ground
    Positional Battle Combined Intermediate Win positional exchanges

    Games by Focus

    Games where the defender is the primary learner:

    Skill Isolation:

    Game What You Learn
    Parry the Straight Parrying mechanics and timing
    Tight Block Absorbing strikes on guard
    Slip the Straight Head movement for straights
    Evade the Punch Punch recognition and evasion selection

    Open Space:

    Game What You Learn
    Close-Range Defense Layering defensive solutions
    Takedown Defense Sprawl, whizzer, recovery
    Clinch Denial Frames, hand fighting, space

    Wall:

    Game What You Learn
    Wall Escape Breaking pins and creating space

    Ground:

    Game What You Learn
    Ground Escape Escaping dominant positions
    Guard Recovery Retaining and recovering guard
    Ground to Standing Wrestling up from bottom

    Games where the attacker is the primary learner:

    Skill Isolation (Weapon Isolation):

    Game What You Learn
    Lead Hand Offense Jab, lead hook, lead uppercut
    Rear Hand Offense Cross, rear hook, rear uppercut
    Lead Leg Offense Lead kicks (teep, roundhouse, etc.)
    Rear Leg Offense Power kicks (roundhouse, body, head)

    Open Space:

    Game What You Learn
    Land the Target Reading defense, setup, target selection
    Sustained Offense Maintaining rhythm through resistance
    Pressure to Takedown Chaining strikes to takedowns
    Pressure to Wall Driving to wall for damage

    Wall:

    Game What You Learn
    Wall Control Establishing and maintaining pins
    Wall Pin to Ground Controlled transitions to ground
    Wall Grinding Damage from wall position

    Ground:

    Game What You Learn
    Ground Access Passing guard, advancing position
    Ground Control Maintaining dominant position
    Ground Finish GnP and submissions from top

    Games where both sides have active learning objectives:

    Open Space:

    Game What You Learn
    Touch and Don't Get Touched Engagement literacy, counter-touching
    Winning the Circle Pressure vs. evasion
    Boundary Trigger Recognizing domain change moments
    Open Space Takedown Takedowns and scrambles

    Wall:

    Game What You Learn
    Stand-Up Loop Positional cycling near the wall

    Ground:

    Game What You Learn
    Positional Battle Winning positional exchanges

    Full MMA Expression

    Every game has a highest level called Full MMA Expression where cross-domain threats are added:

    Game Type MMA Complication Added
    Striking games Grappling threats (shoot, clinch)
    Wrestling games Striking threats
    Ground games Strikes from position

    This ensures skills are tested under realistic MMA pressure. See Full MMA Expression for details.

    Progression Pathways

    Striking Track

    Defensive Skill Isolation → Close-Range Defense → Touch Game
Offensive Skill Isolation → Land the Target → Sustained Offense

    Space Control Track

    Touch Game → Winning the Circle → Boundary Trigger → Wall/Takedown games

    Wall Track

    Pressure to Wall → Wall Control → Wall Grinding (damage path)
                               → Wall to Ground → Stand-Up Loop (ground path)

    Ground Track

    Offensive: Ground Access → Ground Control → Ground Finish
Defensive: Ground Escape → Guard Recovery / Ground to Standing
Combined:  Positional Battle
