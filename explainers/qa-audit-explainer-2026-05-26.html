The false
alarm, explained

Finding #3 assumed every link that reads ../games/x.md in source is broken. That is true for raw HTML links, which MkDocs passes through untouched. It is false for markdown links, which MkDocs resolves against the source file and rewrites. The same file uses both styles, which is what tricked the reviewer.
